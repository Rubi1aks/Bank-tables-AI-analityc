# Помощник по планированию доходов

Приложение для анализа и прогнозирования доходов по регионам. Позволяет загружать данные из Excel, строить граф бизнес-модели, генерировать сценарии прогнозов, находить аномалии и получать новости по теме загруженных данных.

## Запуск проекта

Перед запуском необходимо создать в python-service файл .env со строкой:
AUTH_TOKEN=ваш_токен_гигачата
Без него не будет работать аналитика новостей через GigaChat.

### Быстрый запуск

В корне проекта находится файл RUN.bat. Он позволяет запускать все сервисы одной командой.

При первом запуске на машине первоочередно нажмите пункт 4. (Установить все зависимости).
Далее - запускайте проект с помощью пунктов 2 или 1 (с отладкой). 

### Запуск вручную

#### Backend (Java Spring Boot)
* cd backend
* gradlew bootRun
* Порт: http://localhost:8080
* Swagger: http://localhost:8080/swagger-ui.html

#### Python-сервис (FastAPI)
* cd python-service
* pip install -r requirements.txt
* python app.py
* Порт: http://localhost:5000

#### Frontend (React + Vite)
* cd frontend
* npm install
* npm run dev
* Порт: http://localhost:5173

## Структура документации

В папке Документация находятся файлы:

* Презентация, созданная для защиты продукта. В ней очень кратко упоминаются все члены команды и описывается бизнес-идея. 
* Вся документация собрана в одном файле. В нём описана архитектура системы, перечислены все методы и эндпоинты, приведена структура БД и даны пояснения по ключевым решениям.
* Отчёт о работе - описание лично моей работы со скриншотами работы продукта. 
* Тестовые данные - пример Excel-файлов для демонстрации. 

## Что умеет система

* Загрузка Excel с историческими данными (формат: период, регион, показатель, значение).
* Построение графа бизнес-модели - зависимости между показателями с операторами (+, -, *, /, %).
* Генерация сценариев прогнозов на основе нескольких ML-моделей (SARIMAX, Prophet, Ridge и другие).
* Поиск аномалий во временных рядах с возможностью коррекции.
* Новостная лента с фильтрацией по теме загруженных данных и оценкой влияния через GigaChat.
* Карта регионов, графики и таблицы с виртуализацией для больших объёмов данных.

## Стек технологий

* Backend: Java 17, Spring Boot 3, JPA, H2 Database, WebSocket.
* Python-сервис: FastAPI, scikit-learn, statsmodels, Prophet, GigaChat SDK.
* Frontend: React 18, TypeScript, Vite, Tailwind, Zustand, Recharts, React Flow.
